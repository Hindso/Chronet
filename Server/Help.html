This is the help page, where you can find information useful for playing on our server.

  • Community resources
    • Chronet Roleplay Wikia (WIP)
      • A wiki created by players, includes guides and information on a variety of topics.
    • Chronet Forums
      • The sub-forum where you can find player-created guides.
    • Chronet Lite Roleplay Guide (OLD)
      • An extensive guide covering every aspect of playing on our server.
  • Keyboard shortcuts
    • F1 - Help menu
      • This will display the webpage you're currently viewing.
    • F2 - Purchase/sell door and trade interface
      • This will buy or sell the door you're looking at, if facing a door, or attempt to initiate a trade, if facing a player.
    • F3 - Quick inventory
      • This may be used to access your inventory, which will pop-up on the right-side of your screen.
      • It can also be used to vote in elections or enter lotteries, as it enables the cursor.
    • F4 - Main menu
      • This may be used to access the main menu, in which you will find tabs such as Help, Market, and Inventory.
      • It can also be used to vote in elections or enter lotteries, as it enables the cursor.
    • C - Context menu
      • Holding this allows access to the VCMod settings.
      • Holding this also allows access to the inventory.
      • It can also be used to vote in elections or enter lotteries, as it enables the cursor.
    • G - Government radio push-to-talk
      • Holding this broadcasts your voice chat over the selected channel, if you have the privileges to talk in your current channel.
    • T - Government radio channel
      • This sets which channel you're tuned into on the radio that all government officials have access to.
    • TAB - Scoreboard
      • This opens the player scoreboard, where you can find people's Steam and roleplay names, and their occupation.
      • Admins can be identified by the yellow shield next to their name on the scoreboard.
      • It can also be used to vote in elections or enter lotteries, as it enables the cursor.
  • Commands
    • Out-of-character (OOC) chat commands:
      • @ - Admin chat
        • This sends a message to all admins currently on the server.
      • // or /ooc - Out-of-character (OOC) chat
        • This sends a message to all players currently on the server.
      • /// or /looc - Local out-of-character (LOOC) chat
        • This sends a message to all players in your vicinity.
    • In-character (IC) chat commands
      • /me - Roleplay actions
        • This is used to describe actions your character is roleplaying.
      • /w - Whisper
        • This is used to talk to players in your immediate proximity.
      • /y - Yell
        • This is used to talk to players at some distance.
      • /pm [partial name] - Private message (PM)
        • This is used to privately communicate to other players.
      • /r - Respond to PM
        • This sends a private message to the last person who sent you a PM.
      • /g - Group chat
        • This sends a message to all players in your job's group (e.g. government officials).
      • /org - Organisation chat
        • This sends a message to all players in your organisation.
      • # - Party chat
        • This sends a message to all players in your party.
      • /911 - Emergency chat
        • This sends an emergency message to all government officials
      • /advert - Advertisements
        • This sends a message to all players on the server, and is used solely to advertise business.
      • /broadcast - Mayor broadcasts
        • This is used by the mayor to send a message to all players on the server.
    • Roleplay commands:
      • /give [amount]
        • This gives a set amount of money to the player you are looking at.
      • /dropmoney [amount]
        • This drops a set amount of money wherever you are looking at.
      • /surrender
        • This makes your character put their hands above their head, and limits your movement.
      • /holster
        • This holsters your current item/weapon, so it is returned to your inventory.
      • /resign
        • This resigns you from your current job.
      • /roll
        • This rolls a random number between 0-100, used to settle roleplay disputes.
    • Police and mayor commands:
      • /wanted [partial name]
        • This makes the player wanted, so police can arrest them on sight.
      • /warrant [partial name]
        • This puts a warrant out on a player, so their property can be searched.
    • Door commands:
      • /buydoor
        • This will buy the door you're currently looking at, if it's available.
      • /selldoor
        • This will sell the door you're currently looking at.
      • /title
        • This allows you to set the title of a door, which will appear when players look at it.
      • /setparty
        • This adds all current party members as owners of the door, so they can lock and unlock it.
    • Miscellaneous commands:
      • /help
        • This opens the page you're currently viewing.
      • /rules
        • This opens the rules page, where all the rules for playing on the server can be found.
      • /content
        • This opens the server's workshop collection page.
      • /motd
        • This opens the front page of the Chronet website.
      • /playtime
        • This prints your current playtime into chat.
  • Admin commands
    • Chat commands:
      • /goto [partial name]
        • This teleports you to the player.
      • /bring [partial name]
        • This brings the player to you.
      • /kick [partial name] [reason]
        • This kicks the player from the server.
      • /slay [partial name] [reason]
        • This kills the player.
      • /freeze [partial name]
        • The prevents the player from moving, turning, or performing any actions.
      • /unfreeze [partial name]
        • This allows a frozen player to behave as normal again.
      • /revive [partial name]
        • This revives the player if they're unconscious.
      • /forcerename [partial name]
        • This forces the player to change their roleplay name.
      • /spectate [partial name]
        • This allows you to observe a player.
      • /warn [partial name] [reason]
        • This files a warning for the player in the warning system.
      • /cleardecals
        • This clears all the decals on the server, such as bloodstains and bulletholes.
    • Console commands (also usable through the admin menu interface):
      • admin_cleanup_dc_players
        • This removes all disconnected players' props.
      • admin_cleanup_fire
        • This extinguishes all ongoing fires.
      • admin_freeze_all_props
        • This freezes all props.
      • admin_switch_off_all_police_radios
        • This stops all ongoing communication on the government radio.
      • admin_remove_recovery [partial name]
        • This removes a player's recovery status.
      • admin_reset_recovery_health [partial name]
        • This removes a player's recovery status and resets their health.
